# Sandbox Lab

Four browser sandboxes about emergence. Vanilla JS, no frameworks, open `index.html` for the dashboard.

| Sandbox | What emerges |
|---|---|
| [evolution-simulator](evolution-simulator/) | Species. Organisms with neural-network brains and mutating genomes evolve hunting and grazing strategies. |
| [gravity-warp](gravity-warp/) | Orbits. Bend spacetime with the cursor and watch n-body choreography form and collapse. |
| [chaos-synth](chaos-synth/) | Music. Strange attractors (Lorenz and friends) drive a web-audio synthesizer. |
| [assembly-biosphere](assembly-biosphere/) | Chemistry. Self-assembling particles form bonds, molecules, and proto-metabolisms. |

Each sandbox runs standalone: open its folder's `index.html`, or `npm run dev` inside evolution-simulator for the Vite version.
